What are the differences between map, filter, forEach, and find ?

1. Map:
The map() method creates a new array with the results of calling a provided function on every element in the original array. It returns a new array with the same number of elements as the original array, but with each element transformed based on the function provided. The original array is not modified.

2. Filter:
The filter() method creates a new array with all elements that pass the test implemented by the provided function. It returns a new array with only the elements that meet the condition specified in the function. The original array is not modified.

3. forEach:
The forEach() method executes a provided function once for each array element. It does not return anything and is used when we want to perform a certain action for each element in the array.

4. Find:
The find() function returns the first element in an array that satisfies a given condition specified in a function. It returns undefined if no element satisfies the condition.
